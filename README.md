# 🛒 Retail Sales Analysis Using PostgreSQL

## 📌 Project Overview

This project analyzes retail sales data using PostgreSQL to uncover meaningful business insights from customer transactions. The analysis focuses on sales performance, customer behavior, product category trends, and transaction patterns through SQL-based data exploration and reporting.

The dataset contains over **2,000 retail transactions** across multiple product categories, including Clothing, Beauty, and Electronics.

---

## 🎯 Objectives

* Import and clean retail sales data.
* Perform exploratory data analysis (EDA).
* Analyze customer purchasing behavior.
* Identify top-performing categories and customers.
* Discover monthly and yearly sales trends.
* Generate actionable business insights using SQL.

## 📊 Business Questions Solved

### 1. Sales on a Specific Date

Retrieved all transactions made on a selected date.

### 2. Clothing Sales Analysis

Identified Clothing transactions with quantities exceeding a specified threshold during November 2022.

### 3. Category-wise Revenue

Calculated total sales generated by each product category.

### 4. Customer Demographics

Determined the average age of customers purchasing Beauty products.

### 5. High-Value Transactions

Retrieved transactions with sales values exceeding ₹1000.

### 6. Gender-wise Transaction Analysis

Analyzed transaction counts by gender and product category.

### 7. Best-Selling Month Analysis

Identified the highest-performing month for each year using SQL window functions.

### 8. Top Customers

Ranked customers based on total sales generated.

### 9. Unique Customer Analysis

Calculated the number of distinct customers in each category.

### 10. Shift-wise Order Distribution

Classified transactions into:

* Morning (< 12 PM)
* Afternoon (12 PM – 5 PM)
* Evening (> 5 PM)

and analyzed order volumes across shifts.

## 📸 Sample Queries

```sql
SELECT category,
       SUM(total_sale) AS net_sales
FROM retail_sales
GROUP BY category;
```

```sql
SELECT customer_id,
       SUM(total_sale) AS total_sales
FROM retail_sales
GROUP BY customer_id
ORDER BY total_sales DESC
LIMIT 5;
```
